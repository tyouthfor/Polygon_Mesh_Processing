# Halfedge data structures

### Reference
* https://github.com/yig/halfedge

半边数据结构的 C++ 实现，无其他依赖，能够遍历顶点的所有 one-ring neighbors。同时在 OpenMesh_example.cpp 中直接调用了 OpenMesh 库提供的 API 进行遍历。
